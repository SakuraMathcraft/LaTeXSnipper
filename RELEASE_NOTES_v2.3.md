# LaTeXSnipper v2.3 发布说明

发布日期：2026-04-05

`v2.3` 是一次面向“自动排版 -> 编译 PDF -> 对照修订”工作流的重点更新。相比 `v2.2.1`，本版将自动排版窗口升级为可长期高频使用的 PDF 联动编辑环境，并对渲染性能与交互细节做了系统性收敛。

## 本版重点：自动排版编译 PDF

- 自动排版窗口支持稳定的 PDF 编译预览闭环。
  - 源码区（TeX）编辑后可直接编译并刷新右侧 PDF。
  - 编译状态、失败提示、导出流程更完整。
  - 后端选择（Poppler/Fitz）可在同一窗口内切换。
- 源码与 PDF 双向联动增强。
  - PDF 位置可反向定位到源码行（SyncTeX edit）。
  - 源码行可正向跳转到 PDF 对应位置（SyncTeX view）。
  - 新增源码右键菜单：剪切 / 复制 / 粘贴 / 跳转到 PDF。
- 跳转高亮反馈可视化。
  - 源码跳 PDF 后，目标位置显示蓝色高亮带。
  - 高亮带支持自动消失，并可随滚动保持锚定，不再漂移。
  - 高亮宽高与位置计算已针对行内范围与基线做对齐优化。

## 相比 v2.2.1 的性能优化

- Poppler 预览性能优化。
  - 可见页优先缓存刷新，降低无关页开销。
  - 缺页批量 SVG 生成，减少命令调用频率。
  - 动态预取半径（随滚动速度变化）。
  - 动态 LRU 上限（结合视口规模和刷新耗时）。
- 放大镜渲染链路优化（Poppler/Fitz）。
  - 单阶段高清渲染，消除 fast/hq 切换闪烁。
  - 预测复用帧保留，用于减轻拖动空窗感。
  - 高刷自适应交互时序保留。
  - 边界命中与中心限位统一，边缘/角落行为一致。
- 网格预览能力（自动排版窗口）。
  - 新增网格模式（1x1 / 2x1 / 3x1）。
  - 放大场景下列步长自动扩展，避免页面融合重叠。
  - 网格间距收敛，页间距离更接近边界留白。

## 相比 v2.2.1 的小修复

- 放大镜交互稳定性修复。
  - 修复首帧偶发空白。
  - 修复边缘相切卡住。
  - 修复滚动/拖动时高亮或镜体定位偏移。
- PDF 跳转反馈修复。
  - 修复高亮带上下偏移过大。
  - 修复左端漏字与右端超出问题（通过参数收敛）。
- 自动排版窗口可用性修复。
  - 后端切换/关闭的资源收尾更完整。
  - 相关状态重置更彻底，减少跨操作残留。

## 兼容性与说明

- `v2.3` 继续兼容 `v2.2.1` 的主要使用路径与数据格式。
- 推荐在 Windows + TeX Live / MiKTeX 环境下启用完整 SyncTeX 联动。
- 若系统缺少 Poppler 命令链，将自动回退 Fitz 兼容路径。

## 升级建议

- 若你主要做文档级推导与修订，优先使用自动排版窗口作为主编辑入口。
- 若你需要多页并排审阅，建议使用 `2x1` 或 `3x1` 网格。
- 若你依赖源码/PDF 定位，请先完成一次编译，确保 SyncTeX 文件为最新。
