LaTeXSnipper — Help

Edit, recognize, and insert mathematical equations into Office documents.

Getting Started

Before using the add-in, start the LaTeXSnipper bridge application on your computer. The add-in auto-connects on launch. If needed, click Connect in the sidebar to refresh the connection.

Ribbon Commands

Editor

Opens the full equation editor dialog with symbol panel, visual math editing, and LaTeX source view.

Insert Formula

Converts the sidebar LaTeX into an editable OMML equation at the cursor position. Use the sidebar options to set Display mode (centered, own line) and numbering.

Auto Numbered

Adds automatic numbering to the currently selected LaTeXSnipper equation. The equation becomes a centered display equation with an auto-number. After numbering, the sidebar resets to its initial state.

Screenshot OCR

Opens LaTeXSnipper recognition tools. Capture a screen region containing a formula; the recognized LaTeX appears in the sidebar. Then edit and insert it into the document.

Editing Equations

Load Selected

Click on a LaTeXSnipper equation in the document, then click Load Selected to open it in the editor dialog. The sidebar also refreshes to show the equation's LaTeX, display mode, and numbering.

Delete Selected

With an equation selected, click Delete Selected to remove it. Numbered equations are removed together with their layout table.

Renumber All

Renumbers every LaTeXSnipper numbered equation in document order (1), (2), (3)... Also normalizes table layout and alignment for all numbered equations.

Equation Numbering

  • Auto number — Automatically assigns the next sequential number (n) when inserting.
  • Manual # — Type a custom number (e.g. (5), A.1). Auto number is disabled when a manual value is entered.
  • All numbered equations are display equations in a centered table. Renumber updates them in document order.

Settings

  • Bridge URL — Address of the LaTeXSnipper bridge (default http://127.0.0.1:8765). Change if the bridge runs on a different machine or port.
  • Bridge token — Session token auto-filled when you connect. Required for secure communication with the bridge.
