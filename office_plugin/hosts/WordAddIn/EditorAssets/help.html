LaTeXSnipper Office 插件

在 Office 内插入、编辑和维护由 LaTeXSnipper 管理的原生公式。

快速开始

1. 启动客户端

打开 LaTeXSnipper 桌面端，并在桌面端设置中启用 Office 插件功能。插件通过本机 Bridge 与桌面端通信。

2. 编写公式

在侧边栏输入 LaTeX。截图识别结果也会写回侧边栏；如果编辑器窗口已存在，会同步刷新编辑器内容。

3. 插入到文档

Ribbon 提供行内公式、行间公式和带编号公式三个入口；侧边栏插入按钮使用侧边栏当前选项。

如果发送请求失败，请确认 LaTeXSnipper 桌面端正在运行，并且已经开启 Office 插件功能。

核心功能

原生公式插入

插件会把当前 LaTeX 通过本机 Bridge 转换为 Word 原生 OMML 公式。行内公式跟随正文排版，行间公式单独居中显示，带编号公式使用表格布局保存公式和编号。

加载、更新和删除

选中由 LaTeXSnipper 插入的公式后，可以加载到编辑流程中修改。更新会保留该公式的管理元数据；删除会删除公式本体以及对应的元数据。

截图识别

点击截图识别后，插件等待下一次 LaTeXSnipper 截图结果。可以使用桌面端全局快捷键截图；等待过程中再次点击截图识别会取消当前请求。

自动编号和重编号

带编号公式支持自动编号。重编号会按照文档顺序更新自动编号公式的编号文本和元数据，不重排公式内容。默认编号位置可以在设置中选择左编号或右编号。

插件架构

项目当前实现
运行方式Windows 桌面 Office 原生插件，Ribbon、侧边栏、设置和帮助页都在 Office 内打开。
公式写入在 Word 中写入原生 OMML，并保存 LaTeXSnipper 管理元数据。
连接方式通过本机 Bridge 连接 LaTeXSnipper 桌面端，默认地址为 127.0.0.1:28765。
维护目标优先提供稳定的 Word 桌面体验，并为 PowerPoint 和 OLE 公式对象保留清晰扩展边界。

边界说明

  • 只有由 LaTeXSnipper 插入并保留元数据的公式，才能被插件加载、更新、删除、自动编号和重编号。
  • 普通 Word 原生公式没有 LaTeXSnipper 元数据，插件不会猜测其 LaTeX 来源。
  • 复制粘贴后丢失元数据的公式，不能作为受管理公式继续维护。
  • 如果 Word 禁用了加载项，请关闭 Word，重新运行插件注册脚本，再打开 Word 测试。

LaTeXSnipper Office Plugin

Insert, edit, and maintain native Office formulas managed by LaTeXSnipper.

Quick Start

1. Start the client

Open the LaTeXSnipper desktop client and enable the Office plugin feature. The plugin talks to the client through the local Bridge.

2. Prepare LaTeX

Type LaTeX in the side pane. Screenshot OCR also writes back to the side pane; if the editor window already exists, it is updated too.

3. Insert into Word

The Ribbon provides separate commands for inline, display, and numbered formulas. The side pane insert button uses the side pane options.

If a request cannot be sent, make sure the LaTeXSnipper desktop client is running and the Office plugin feature is enabled.

Features

Native formula insertion

The plugin converts the current LaTeX through the local Bridge and writes native Word OMML. Inline formulas flow with text, display formulas are centered, and numbered formulas keep formula content and number metadata together.

Load, update, and delete

LaTeXSnipper-managed formulas can be loaded for update. Updates preserve metadata; deletion removes both the formula and its metadata.

Screenshot OCR

Screenshot OCR waits for the next LaTeXSnipper capture. Use the desktop global shortcut to select a formula region; click Screenshot OCR again while waiting to cancel.

Automatic numbering

Numbered formulas support automatic numbering. Renumbering updates number text and metadata in document order without re-rendering formula content. The default number side can be changed in Settings.

Boundaries

  • Only formulas inserted by LaTeXSnipper with metadata can be loaded, updated, deleted, automatically numbered, and renumbered.
  • Plain Word equations do not contain LaTeXSnipper metadata, so the plugin does not infer their LaTeX source.
  • Copied formulas that lost metadata cannot continue as managed formulas.
  • If Word disables the add-in, close Word, run the registration script again, then reopen Word.
